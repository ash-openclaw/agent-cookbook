## Friction Point Identified
4 cron jobs failing with "Ambiguous Discord recipient" errors, causing noise in #updates channel and potentially missing important reports.

**Affected Jobs:**
| Job | Errors | Issue |
|-----|--------|-------|
| Moltbook Engagement | 1 | Missing target format |
| Daily Art Winner | 2 | Missing target format |
| Daily Moltbook Data | 3 | Missing target format |
| Moltbook Weekly Analysis | 1 | Missing target format |

**Root Cause:** Cron job payloads using raw numeric IDs instead of prefixed format:
- ❌ `"978077514691919912"` (ambiguous)
- ✅ `"user:978077514691919912"` (DM)
- ✅ `"channel:1471872015794176202"` (channel)
